
Income Prediction Analysis
==========================

Description
-----------
This project involves exploring and analyzing the 'adult' dataset to predict whether an 
individual earns above 50K a year. The analysis encompasses exploratory data analysis 
(EDA) and logistic regression modeling.

Dependencies
------------
- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- python-docx

Instructions
------------
1. Ensure all dependencies are installed.
2. Download and place the 'adult.csv' dataset in the working directory.
3. Execute the Python script ‘income_prediction_analysis.py’ to perform the analysis.
4. Review the generated plots and metrics for insights.

Files
-----
- `income_prediction_analysis.py`: Contains the Python code for the analysis.
- `income_prediction_report.docx`: Generated report detailing the analysis and findings.
